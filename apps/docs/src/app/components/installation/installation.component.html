Getting started

Installation

Install the library using

npm install pro-widgets

Then you can either import all of the Widgets by importing the WidgetsModule. 

import { WidgetsModule } from 'pro-widgets'

Or you can import each module separately.

import { GaugeModule } from 'pro-widgets'

Quick start

If you have imported the module successfully to your application you should be able to create the gauge (or analog-stick) simply by 

<pro-gauge></pro-gauge>
<pro-analog-stick></pro-analog-stick>

In the gauge section you can customize and live-preview all the input parameters and copy and paste the code output. Same goes for the analog stick.
