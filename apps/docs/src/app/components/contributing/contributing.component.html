Contributing

All forms of contributions are welcome. Hand in widget ideas, bug reports, or feature requests. If there are more then me contributing to this lib, I consider creating a contributors section.

Developer Guidelines

If you want to create a new widget create a feature request first. If it is a widget request:
  1. Create the widget in the /projects/pro-widgets/src/lib/WIDGETID
  2. The widget has to be a component containing the SVG in the template
  3. Create a module for the widget which declares and exports the component
  4. Get inspirations about data-bindings from the existing widgets
  5. Create meaningful unit tests (some are missing for existing components yet)
  6. Export the widget module in the ./pro.module.ts
  7. Export the component and the module in the public_api.ts
Creating docs for the widget:
  1. run run npm build:widgets
  2. Create an entry in the ./src/app/data/widgets/ folder (orientate on existing ones)
  3. Create at least one theme for your widget in ./src/app/data/themes
